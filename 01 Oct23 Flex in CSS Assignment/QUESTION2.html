  • Justify content:-The justify-content property in CSS is used to align the items of a flex container or a grid container along the main axis (horizontally) or the inline axis12. It defines how the browser distributes the space between and around the content items. For example, you can use justify-content: center to pack the items in the center of the container, or justify-content: space-between to distribute the items evenly with the first and last items flush with the edges of the container

  • align-itemsThe align-items property in CSS is used to specify the default alignment for items inside a flexbox or grid container . In a flexbox container, the flexbox items are aligned on the cross axis, which is vertical by default opposite of flex-direction In a grid container, the grid items are aligned in the block direction . The align-items property accepts the following values
    properties of align items:- start,end,baseline,center.

  • Gap:-The gap property in CSS is used to specify the size of the gap between rows and columns in a grid, flexbox, or multi-column layout . It is a shorthand for the row-gap and column-gap properties . The gap property accepts two values, separated by a space. The first value specifies the size of the gap between rows, while the second value specifies the size of the gap between columns . The default value of the gap property is normal, which means that no gap is applied

  • flex-directionThe flex-direction property in CSS is used to specify the direction of the main axis of a flex container and the order of the flex items . The flex-direction property accepts the following values:


    properties of flex-direction:- row,row reverse, column,column reverse.

  • Flex-wrap:-The flex-wrap property in CSS is used to specify whether the flexible items inside a flex container should wrap or not . The flex-wrap property accepts the following values


    properties of flex wrap:- nowwrap,wrap and wrapreverse.
